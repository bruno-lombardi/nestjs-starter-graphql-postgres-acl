## Ecommerce GraphQL Application with NestJS

Built with [Nest](https://github.com/nestjs/nest), a TypeScript based NodeJS Web framework.

## Installation

```bash
$ npm install
# or
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
